# Raw Data for Supplementary Figures 1–5

This folder contains raw datasets used to generate Supplementary Figures 1–5 in the accompanying supplementary material.

## File Organization

Subdirectories or files may be organized by supplementary figure number. Each dataset should include:
- Raw recording data (`.mat`, `.bin`, or other formats as applicable)
- Preprocessed intermediate files
- Associated metadata

## Relationship to Main Figures

Supplementary figures typically provide additional analyses, replicates, or alternative views of data supporting the main manuscript claims. See the main `README.md` and `matlab/figures/` documentation for context.
